# InternetSaleAds

Функция: Показ и сортировка объявлений, добавление, поиск, удаление и модерация 
    Для того, чтобы проверить функциональность программы
    В качестве Пользователя взаимодействия с системой
    Я хочу выводить на экран и сортировать, добавлять, искать, удалять и модерировать объявления
	
	Бэкграунд: 
			Дано: Консольное приложение
			И: <кнопки> с выбором <меню>
			
			Примеры:
				| Кнопка | Меню  |
				|--------|-------|
				| 1      | Админ |
				| 2      | Юзер  |
				| 0      | Выход |
				
	Бэкграунд: 
			Дано: Консольное приложение
			И: <кнопки> с выбором <меню>
			
			Примеры:
				| Кнопка |             Меню             |
				|:------:|:----------------------------:|
				| 1      | Показать список объявлений   |
				| 2      | Добавить объявление          |
				| 3      | Поиск объявления по названию |
				| 4      | Удалить объявление           |
				| 5      | Объявления на модерации      |
				| 0      | Выход                        |
				
	Сценарий: Показ списка объявлений
		Дано: Консоль выводит список <кнопок> с соответствующим им <меню>
		Когда: Пользователь жмет кнопку 1
		То: Консоль выводит список вида <наименование> <описание> <цена> <номер тел.> <продавец> <дата размещения>
		
			Примеры:
				| Наименование | Описание     | Цена | Номер тел.  | Продавец | Дата размещения     |
				|--------------|--------------|------|-------------|----------|---------------------|
				| Консоль      | PayStation $ | 3000 | 89000002222 | Петя     | 24.04.2021 23:19:34 |
				| Телевизор    | Pony         | 5000 | 89000003333 | Оля      | 24.04.2021 23:19:34 |
				
	Сценарий: Сортировка объявлений
		Дано: Консоль выводит список <кнопок> с выбором меню <сортировать>
		Когда: Пользователь жмет <кнопку>
		То: Консоль выводит <результат>
		
			Примеры:
				| Кнопка | Сортировать | Результат |
				|--------|-------------|-----------|
				| 1      | по цене     | 0 < uint  |
				| 2      | по имени    | A-Z       |
				| 3      | по дате     | <         |
				| 0      | выход       | выход     |
				
	Сценарий: Фильтрация объявлений
		Дано: Консоль выводит список <кнопок> с выбором меню <сортировать>
		Когда: Пользователь жмет <кнопку> 4
		И: Вводит <номер тел.>
		То: Консоль выводит <результат>
		
			Примеры:
				| Кнопка | Номер тел.  | Результат             |
				|--------|-------------|-----------------------|
				| 4      | 89000001111 | 4 объявления          |
				| 4      | 3435        | Объявлений не найдено |
				| 4      | ваправр     | Ошибка!               |
				
	Сценарий: Добавление нового объявления
		Дано: Консоль выводит список <кнопок> с соответствующим им <меню>
		Когда: Пользователь жмет кнопку 2
		И: Вводит <наименование> <описание> <цена> <номер тел.> <продавец>
		То: Консоль выводит <ответ>
		
			Примеры:
				| Наименование | Описание     | Цена  | Номер тел.  | Продавец | Ответ                   |
				|--------------|--------------|-------|-------------|----------|-------------------------|
				| Консоль      | PayStation $ | 3000  | 89000002222 | Петя     | Объявление добавлено    |
				| Телевизор    | Pony         | 10000 | 89000003333 | Оля      | Объявление на модерации |
				| dgfhdfg      | dfdfghfgh    | -45   |             |          | Ошибка!                 |
				| 234543634    | 657аера      | 354   | чиаьпрьапт  |          | Ошибка!                 |
		
	Сценарий: Поиск объявления по названию
		Дано: Консоль выводит список <кнопок> с соответствующим им <меню>
		Когда: Пользователь жмет <кнопку> 3 и вводит <название> объявления
		То: Консоль выводит <результат>
		
			Примеры:
				| Название | Результат    |
				|----------|--------------|
				| Кактус   | 2 объявления |
				| Телефон  | 1 объявление |
				| dhfghf   | Ошибка!      |
				|          | Ошибка!      |

	Сценарий: Удаление объявления
		Дано: Консоль выводит список <кнопок> с соответствующим им <меню>
		Когда: Пользователь жмет <кнопку> 4 и вводит <номер> объявления
		То: Консоль выводит <ответ>
		
			Примеры:
				| Номер | Ответ               |
				|-------|---------------------|
				| 1     | Объявление удалено! |
				| 5     | Объявление удалено! |
				| fdg   | Ошибка!             |
				|       | Ошибка!             |

	Сценарий: Модерация объявлений
		Дано: Консоль выводит список <кнопок> с соответствующим им <меню>
		Когда: Пользователь жмет <кнопку> 5, вводит <номер объявления>
		И: Вводит <номер меню>
		То: Консоль выводит <ответ>
		
			Примеры:
				| Номер объявления | Номер меню | Ответ                   |
				|------------------|------------|-------------------------|
				| 1                | 2          | Объявление удалено!     |
				| 5                | 1          | Объявление опубликовано |
				| fdg              |            | Ошибка!                 |
				|                  | 0          | Выход                   |
		
	Сценарий: Выход
		Дано: Консоль выводит список <кнопок> с соответствующим им <меню>
		Когда: Пользователь жмет кнопку 0
		То: Консоль закрывается
